# Cannabis Price Tracker

Real-time cannabis flower pricing across U.S. legal markets. Updated monthly with official state regulatory data.

## Live Site

**[prices.dankreports.com](https://prices.dankreports.com)**

## Features

- **8 State Markets Tracked**: California, Colorado, Illinois, Maine, Maryland, Massachusetts, Michigan, Oregon
- **Official Data Sources**: All prices sourced directly from state cannabis regulatory agencies
- **Monthly Updates**: Fresh data published as states release new market reports
- **Interactive Visualization**: Color-coded price ranges and sortable data tables
- **Mobile Responsive**: Optimized for desktop and mobile viewing

## Price Range (Current)

- **Lowest**: California ($1.40/gram)
- **Highest**: Maryland ($8.28/gram)
- **Median**: $4.13/gram

## Data Sources

Each state's pricing data links directly to official regulatory dashboards:

- **California**: [DCC Data Dashboard](https://www.cannabis.ca.gov/resources/data-dashboard/)
- **Colorado**: [MED Dashboard](https://public.tableau.com/app/profile/cu.business.research.division/viz/ColoradoMEDDashboard/Overview)
- **Illinois**: [Cannabis Sales Figures](https://cannabis.illinois.gov/research-and-data/sales-figures.html)
- **Maine**: [OCP Retail Sales Data](https://www.maine.gov/dafs/ocp/open-data/adult-use/retail-sales)
- **Maryland**: [MCA Data Dashboard](https://cannabis.maryland.gov/pages/data-dashboard.aspx)
- **Massachusetts**: [CCC Open Data](https://masscannabiscontrol.com/open-data/data-catalog/)
- **Michigan**: [CRA Statistical Reports](https://www.michigan.gov/cra/resources/cannabis-regulatory-agency-licensing-reports/cannabis-regulatory-agency-statistical-report)
- **Oregon**: [OLCC Market Data](https://www.oregon.gov/olcc/marijuana/pages/marijuana-market-data.aspx)

## Technology

- Pure HTML/CSS/JavaScript (no build process)
- Embedded data structure for simple monthly updates
- Responsive grid layouts with CSS animations
- Zero dependencies

## Update Schedule

Data is updated monthly as new state reports are published. Most states publish between the 5th-15th of each month for prior month data.
